JavaScript Variable Practice

Both let and var are used to declare variables in JavaScript. They differ in their scope.

let
Scope: Immediate enclosing block denoted by {}
var
Scope: Immediate function body

let Example:

                    let city = 'San Diego';
    
                    function welcomeCity() {
                        alert(`Welcome to ${city}!`);
                    }
    
                    welcomeCity();

As always, don't forget to Cmd + s to save your code as you work!
